# MEALFLOW

MealFlow is a recipe management and meal planning application. It provides a platform for managing personal recipes, scheduling weekly meals, and generating automated shopping lists.

## Features

- Recipe Management: Users can upload, edit, and search their own private recipes.
- Meal Planning: Users drag-and-drop their recipes onto a weekly calendar to plan meals.
- Shopping List: Automatically generates a consolidated shopping list based on the planned meals.
